# No. 2 Batabari Youth Club Management System

A role-based web application for managing members, loans, and monthly collections.

## Features
- Admin dashboard to manage members and loans
- Member portal to view loan details and collection history
- Paid and pending month tracking
- Interest calculation with real-time loan status
- Fully responsive UI

## Tech Stack
- React
- Tailwind CSS
- shadcn/ui
- Supabase
- TanStack React Query
